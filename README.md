## 项目介绍
基于springboot的音乐管理系统：前端 html、jquery、bootstrap，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成音乐在线播放、收藏、音乐搜索等功能于一体的系统。

## 功能介绍

- 基本功能：登录，注册，退出，密码修改
- 网站首页：主导航栏，全局搜索，轮播图，歌单推荐，巅峰排行
- 在线播放：音乐列表，音乐详情，播放，暂停
- 我的音乐：收藏音乐列表，收藏取消
